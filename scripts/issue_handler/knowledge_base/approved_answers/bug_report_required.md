# Bug Report Guidelines

Thank you for taking the time to report a bug! To help us investigate and fix the issue, please include the following information:

## Required Information

1. **Steps to reproduce**: What exactly did you do before the issue occurred?
2. **Expected behavior**: What did you expect to happen?
3. **Actual behavior**: What actually happened instead?
4. **Environment**:
   - Device model (e.g., SoundTouch 300, SoundTouch 10)
   - Operating system (e.g., Windows 11, macOS 14, Ubuntu 24.04)
   - Browser (e.g., Chrome 124, Firefox 126)
   - OpenCloudTouch version or Docker image tag

## Helpful Extras

- Screenshots or screen recordings
- Browser console logs (F12 → Console tab)
- Docker logs: `docker logs opencloudtouch`

## Bug Report Template

You can use our [bug report template](https://github.com/opencloudtouch/opencloudtouch/issues/new?template=bug_report.yml) for a structured report.

The more details you provide, the faster we can identify and fix the issue!
